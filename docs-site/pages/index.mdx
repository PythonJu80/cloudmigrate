# CloudFabric Documentation

CloudFabric is a unified cloud management platform for multi-cloud infrastructure operations.

## Platform Services

| Service | Purpose |
|---------|---------|
| **CloudMigrate** | File transfers to S3, GCS, Azure Blob, and OCI Object Storage with progress tracking, checksums, and transfer history |
| **CloudFlow** | Visual workflow builder with 140+ nodes across AWS, GCP, Azure, and Oracle. Supports triggers (manual, cron, webhook), AI nodes, and local file operations |
| **CloudArch** | Architecture diagram designer with 50+ AWS services. AI-assisted design via natural language. Generates CloudFormation templates for deployment |
| **CloudWatch** | Real-time monitoring pulling live data from AWS CloudWatch, CloudTrail, and Cost Explorer APIs |
| **CloudGPT** | AI assistant with RAG-powered context from your company knowledge base and transfer history |

## Supported Cloud Providers

- **AWS** - Full support including S3, EC2, RDS, Lambda, and 50+ services
- **Google Cloud** - Storage, Compute, Cloud SQL, Cloud Functions
- **Azure** - Blob Storage, VMs, SQL Database, Functions
- **Oracle Cloud** - Object Storage, Compute, ATP/ADW
