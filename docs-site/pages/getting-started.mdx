# Getting Started

## Create an Account

1. Go to the registration page
2. Enter name, email, and password
3. Check email for verification link
4. Click link to verify and activate account
5. Sign in

A tenant is automatically created for your account.

## Configure AWS

At minimum, configure AWS to use CloudMigrate and CloudWatch:

1. Go to **Settings** from the sidebar
2. Select **AWS** tab under Cloud Providers
3. Enter:
   - Access Key ID
   - Secret Access Key
   - Region
4. Click **Test Connection**
5. Click **Save**

## Upload Your First File

1. Go to **CloudMigrate** from the sidebar
2. Click the **Upload** tab
3. Select a bucket from the dropdown
4. Drag a file onto the upload zone
5. Watch the progress bar
6. Check the **History** tab to see the transfer record

## View Monitoring Data

1. Go to **CloudWatch** from the sidebar
2. If AWS is configured, you'll see:
   - EC2 CPU metrics
   - Lambda error rates
   - Cost data
3. If not configured, you'll see a prompt to add credentials

## Try CloudGPT

1. Go to **CloudGPT** from the sidebar
2. Ask a question like:
   - "Show me a chart of transfers by status"
   - "What's in my S3 bucket?"
3. View the generated visualization or response

## Build a Workflow

1. Go to **CloudFlow** from the sidebar
2. Drag a **Manual Trigger** onto the canvas
3. Drag an **S3 List Objects** node
4. Connect them
5. Configure the S3 node with a bucket name
6. Click **Run Flow**

## Design Architecture
